Bilder hinzufügen

Mit diesem Formular, können Sie eines oder mehrere Bilder hinzufügen.

Allgemeines
Wenn Sie nur ein Bild eintragen wollen, können sie die restlichen Felder einfach leer lassen. Lansuite unterstützt die Bildformate jpg, png und gif. Wenn Sie Ihr Bild oder Ihre Bilder entsprechend eingetragen haben, müssen Sie anschließend auf den "Eintragen"-Knopf am Ende des Formulars klicken, um Ihre Bilder auf den Lansuite Server hochzuladen, und sie somit für andere User sichtbar zu machen.
Beschreibung

Für jedes Bild müssen Sie eine kurze Beschreibung eintragen, z.B. wann das Bild erstellt wurde.

Datei
Hier können Sie entweder direkt den Pfad des Bildes angeben, oder mit dem Knopf "Durchsuchen" nach dem Pfad des Bildes suchen.
