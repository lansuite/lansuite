Serverdetails

Hier sehen Sie dei Details des aktuell ausgewählten Servers.

Allgemeines

Mit dem Knopf "zurück" gelangen Sie zur Hauptübersicht aller Server zurück.
Mit dem Knopf "ändern" können Sie, sofern Sie der Besitzer dieses Server oder ein Administrator sind, die Serverdetails ändern.
Mit dem Knopf "löschen" können Sie, sofern Sie der Besitzer dieses Server oder ein Administrator sind, den Server wieder aus der Liste, der in Lansuite gespeicherten Server, entfernen.

Name / Besitzer / Servertyp
Hier sehen Sie, welchem User der Server gehört, den Namen des Servers und den Dienst, der auf ihm bereitgestellt wird.
Status / IP-Adresse / Port

Hier sehen Sie den Status des Dienstes ( erreichbar oder nicht erreichbar), die IP-Adresse und den dazugehörigen Port.

Betriebssytem

Hier sehen Sie, welches Betriebssytem auf dem Server installiert ist.

CPU / RAM / HDD

Hier sehen Sie die wichtigsten teschnichen Daten des Servers.

Passwort

Hier sehen Sie, ob der Dienst, der auf dem Server aktiv ist, durch ein Passwort geschützt wird, oder nicht.

Beschreibung

Hier sehen Sie eine kurze Beschreibung des Servers bzw. des Dienstes.
