Moduleinstellungen

Mit diesem Formular, können Sie verschiedene Module von Lansuite aktivieren bzw. deaktivieren.

Allgemeines

Hier sehen Sie alle, in Lansuite verfügbaren Module, diese können Sie hier aktivieren oder deaktivieren. Wenn in dem Kästchen neben dem Modul ein Häckchen angezeigt wird, ist dieses Modul aktiviert.
