Benutzer hinzufügen

Hier können Sie einen Benutzer hinzufügen.

Benutzername

Hier legen Sie, den vom User gewünschtend Namen fest.

Typ

Hier legen Sie fest, ob der neue Benutzer einen "Benutzer", einen "Administrator", oder einen "Operator" Status erhalten soll.

Ein Benutzer kann keine Script oder LAN bezüglichen Daten ändern.
Ein Administrator kann LAN bezügliche Daten ändern, jedoch keine veränderungen in Lansuite selbst bewirken.
Ein Operator hat die "uneingeschränkte Macht" über Lansuite und seine Benutzer. Er kann Daten eines Users genauso ändern, wie die Daten der LAN oder die Einstellungen von Lansuite.

Vorname / Nachname

Hier tragen Sie den "REAL-Name" des Benutzers ein.

Zugriffsberechtigung

Wenn der Benutzer Admin oder Operator ist kannst du hier die Berechtigung für Module freischalten.
Der Benutzer kann dann dieses Modul administrieren.
Gruppe auswählen

Wenn du verschieden Gruppen angelegt hast kannst du dem Nutzer hier eine zuweisen.
Preis auswählen

Wenn du verschieden Tickets bietet kannst du hier eines auswählen.
Bezahlt

Hat der Besucher schon bezahlt? Diese angabe macht natürlich nur sinn wen der Benutzer zur Party angemeldet ist.
Angemeldet

Hier kannst du den Nutzer zur aktuelle ausgewählten Party eintragen.
Die aktuell ausgewählte Party siehst du in der Infobox
Email

Hier tragen Sie die Email-Adresse ein, sie dient später auch zum Login.

Vorhandener Clan / Neuer Clan / Clan Passwort / Clan Webseite

Hier kannst du einen Clan auswählen oder eintragen.
Wenn du einen auswählst musst du das festgelegte Passwort angeben. Bei einem neuen kannst du ein Passwort festlegen.
Im Feld Clan Webseite kannst du euere Clanwebseite angeben.
WWCL-ID / NGL-ID

Gib hier, falls vorhanden, deine Nummer ein, damit deine Spielstände nachher gutgeschrieben werden können.
Strasse und Hausnummer / PLZ und Ort

Hier kannst du deine Adresse angeben.
Personalausweisnummer

Gib bitte die Nummer die auf deinem Personalausweis steht an, für die Altersprüfung.
Geburtstag / Geschlecht

Hier gibst du dein Geburtstag und dein Geschlecht an.
Newsletter

Möchtest du über die Party informiert werden? Dann aktiviere diese Kästchen.
