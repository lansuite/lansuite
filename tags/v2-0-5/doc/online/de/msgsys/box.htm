Messenger

Sollten Sie bereits erfahrungen mit Programmen wie "ICQ", dem "MSN Messenger", oder Ähnlichen gemacht haben, so wird es Ihnen sicherlich leicht fallen, die Funktionsweise dieses Messengers zu verstehen.

Über den Messenger ist es möglich, mit allen Benutzern zu chatten. Dabei können auch Nachrichten an Benutzer verschickt werden, die momentan offline sind. Die Nachricht wird dann im System gespeichert und zugestellt, sobald der Benutzer sich wieder einloggt.

Sollten Sie eine Nachricht von einem Benutzer erhalten, der sich nicht in Ihrer "Buddy-Liste" befindet, so wird dieser temporär in der Gruppe "Nicht-In-Liste" angezeigt - kann aber mit einem Klick auf das dahinter befindliche "+"-Symbol in Ihre "Buddy-Liste" hinzugefügt werden.

Sie können Benutzer wieder aus der "Buddy-Liste" entfernen, indem Sie auf das dahinter befindliche "X"-Symbol klicken, und die anschließende Frage mit "Ja" beantworten.

Über den, sich links neben dem "X"-Symbol befindenden Button, ist es möglich sich genaue Details über den jeweiligen Benutzer anzeigen zu lassen.

Das Nachrichten-Symbol vor dem Benutzernamen erlaubt es, das "Message-Fenster" zu öffnen, über das man mit dem jeweiligen Benutzer chatten kann. Blinkt dieses Symbol, so bedeutet dies, dass eine neue, ungelesene Nachricht von diesem Benutzer vorliegt, die im besagten "Message-Fenster" gelesen werden kann.

Die Farbe des Benutzernamens gibts Auskunft über den Status des jeweilige Benutzers. Ist der Name schwarz, so bedeutet dies, dass der Benutzer "online" ist; ist er rot steht dass dafür, dass der Benutzer "offline" ist.
