void d( )

Magic Debugfunction

Calls addvar from Class debug. You can use it 2 Ways. Just pass a Variable or description+Variable. See Example. Caution : for set a Variablecaption like $variable you must User single quote. ('$variable') Example :

  1.  // Just an Variable. Debugoutput generate an automatic Name lika Debugvar_x
  2.  d($config);
  3.  // Or Name plus Variabe. Use single quote
  4.  d('$config', $config);
