Sitzblock anzeigen

Mit Hilfe dieses Sitzblocks haben Sie die Möglichkeit die gewünschten Benutzerdaten zu einem Platz anzuzeigen.

Benutzerdaten anzeigen

Durch den Klick auf einen Sitzplatz werden Ihnen die Benutzerdaten angezeigt.

Benutzer finden

Wenn Sie herausfinden wollen, wo ein bestimmter Benutzer sitzt, nutzen Sie bitte die Gästeliste um den Benutzer anzuzeigen, dort wird in der letzten Spalte ein Link angezeigt, mit dem direkt der Sitzplatz angezeigt werden kann.
