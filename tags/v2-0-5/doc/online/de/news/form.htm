Newsmeldung eintragen/editieren

Mit diesem Formular, können Sie eine Newsmeldung eintragen/editieren. Diese kann dann von allen Benutzern gelesen und kommentiert werden. Sollten Sie mit der Newsmeldung, die Sie erstellt haben, nicht zufrieden sein, so kann diese nachträglich geändert oder gelöscht werden.

Überschrift

Hier können Sie die Überschrift für Ihre Newsmeldung eingeben. Hierfür stehen Ihnen maximal 35 Zeichen zur Verfügung.
Text
Hier können Sie den Text für Ihre Newsmeldung eingeben. Hierfür stehen Ihnen maximal 5000 Zeichen zur Verfügung. Des weiteren können Sie auch Smilies einfügen, sowie verschiedene Schriftgrößen- und Typen wählen.
Priorität
Hier wählen Sie die Priorität, d.h. die Wichtigkeit der Newsmeldung. Bei hoher Priorität wird die Newsmeldung beim Anzeigen farbig unterlegt.
