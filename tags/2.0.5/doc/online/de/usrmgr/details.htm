Userdetails

Hier sehen Sie dei Details des aktuell ausgewählten Users.

Allgemeines

Mit dem Knopf "auschecken" können Sie den aktuell ausgewählten Benutzer von dieser LAN-Party abmelden.
Mit dem Knopf "ändern" können Sie, sofern es Ihr Account ist oder Sie ein Administrator sind, die Details ändern.
Mit dem Knopf "löschen" können Sie, sofern Sie ein Administrator sind, den Benutzer aus der Lansuite-Datenbank entfernen.
Mit dem Knopf "neues Passwort" können Sie, sofern Sie ein Administrator sind, dem aktuell ausgewählten Benutzer ein neues Passwort zuweisen.
