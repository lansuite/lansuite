Messenger

Hier wird angezeigt, welcher Benutzer sich zur Zeit in Ihrer Buddy - Liste befindet.
Durch blinken des Message - Bildes wird Ihnen ein Mitteilungseingang signalisiert.

Benutzer hinzufügen

Hier haben Sie die Möglichkeit einen oder mehrere Benutzer zu Ihrer Buddy - Liste hinzuzufügen.
Message verschicken Um einem Benutzer eine Nachricht zu schicken, klicken Sie bitte auf das Message - Bild. Folgend öffnet sich der Messenger.
