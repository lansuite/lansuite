Troubleticket hinzufügen

Mit diesem Formular, können Sie ein Troubleticket in Lansuite eintragen.

Allgemeines

Ein Troubleticket ist ein Bericht über ein Problem der LAN, auf der Sie sicher gerade befinden. Dieses Troubleticket wird einem Orga mitgeteilt, dieser wird dann versuchen das Problem zu beheben.
Kurze Beschreibung
Tragen Sie hier bitte nur einige Stichworte ein, die das Problem darstellen.
Priorität
Hier legen Sie die Priorität des Problems fest. Ist es nur ein kleines Problem wie z.B. eine lange Warteschlange auf der Toilette, sollte die Priorität "Normal" gewählt werden. Wenn es z.B. einen öfteren Strom- oder Netzwerkausfall gibt, sollte "Kritisch" gewählt werden. Gibt es z.B. keine Rostwürste mehr könnte man durchaus "Hoch" als Priorität festlegen.
Sichtbarkeit (Orga und Administrator only)
Hier können Sie entscheiden, ob das Troubleticket für jeden auf der LAN sichtbar ist, oder nur für Orgas und Administratoren.
Text
Hier können Sie Ihr Problem genau beschreiben.
