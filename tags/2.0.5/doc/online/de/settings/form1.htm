Datenbank

Mit diesem Formular, können Sie Einstellungen zur Datenbank, die Lansuite benutzen soll, vornehmen.

Default-Design

Hier können Sie festlegen, welches Design für jeden User festgelegt wird.
Lanparty-Name
Hier können Sie den Namen ihrer Lanparty eintragen, dieser erscheint dann im Titel von Lansuite.
WWCL-ID
Hier können Sie Die WWCL-ID der Lanparty und des Orgateams eintragen, wenn Sie keine WWCL-ID besitzen, lassen Sie diese beiden Felder ohne Angaben.
Banneraneige
Zur Zeit noch keien Funktion.
