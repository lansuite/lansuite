Teammanager

Mit Hilfe des Teammanagers können Sie Ihre Teams verwalten.

Einzelspieler-Turniere, an denen Sie teilnehmen

Hier sehen Sie eine Lister aller Einzelspieler-Turniere, zu denen Sie sich angemeldet haben. Mit einem Klick auf "Mich abmelden" können Sie sich hier von dem jeweiligen Turnier wieder abmelden. Dies ist jedoch nur möglich, solange sich ein Turnier noch in der Anmeldephase befindet und noch nicht gestartet wurde.
Teams, die Sie erstellt haben
Hier finden Sie eine Liste aller Teams, die Sie erstellt haben. Mit dem Link "Team abmelden" können Sie das Team von dem jeweiligen Turnier abmelden. Dabei werden auch alle Teammitglieder gelöscht. Das Abmelden eines Teams ist nur möglich, solange sich ein Turnier noch in der Anmeldephase befindet und noch nicht gestartet wurde. Mit dem Link "Spieler hinzufügen" können Sie weitere Spieler in Ihr Team aufnehmen. Mit dem Button "kicken" können Sie den entsprechenden Spieler aus Ihrem Team werfen. Das Hinzufügen und Kicken von Teammitgliedern ist auch noch nach dem Start eines Turnieres möglich.
Teams, in denen Sie Mitglied sind
Hier können Sie aus einem Team wieder aussteigen.
