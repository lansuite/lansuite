Message window

The message-window allows you to get in contact with other users.
In the headline of the window you see which user you are chatting with.
It is possible to have multiple such windows opened. To chatt with more than one person.

The window itsselfe is devided into two parts. The upper one, where you can read the received and sent messages, and the lower one, where you can sent messages
In the upper party you see all messages ordered by time, whith the newest at the bottom. With each message you can see the senders name and the time the message was send. The messages woun't be deleted is you close the window.

In the lower part you can write what ever long messages. With the buttons below you can add smilies to your message.
