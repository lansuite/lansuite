Switch eintragen

Mit diesem Formular können Sie einen Switch eintragen. Dieser kann dann später im NOC verwendet werden. Sollten Sie mit den Daten, die Sie eingegeben haben, nicht zufrieden sein, so können diese nachträglich geändert oder gelöscht werden.

Name

Hier können Sie einen Namen für den Switch eingeben. Hierfür stehen Ihnen maximal 30 Zeichen zur Verfügung.
IP-Adresse
Hier können Sie die IP-Adresse Ihres Switches eingeben. Eine IP Adresse hat im allgemeinen 4 Zahlen, durch Punkt getrennt. Keine Zahl darf größer als 255 sein, und die erste muss größer als 1 sein. Beispiel:

192.168.0.1
127.0.0.1
10.3.5.241
Read-Community
Hier tragen sie den Read-Community-String ihres Switches ein. Konsultieren sie hierfür das Handbuch oder das Web-Interface des Switches. Bei den meisten lautet er Standartmäßig "public".
Write-Community
Hier tragen sie den Write-Community-String ihres Switches ein. Konsultieren sie hierfür das Handbuch oder das Web-Interface des Switches. Bei den meisten lautet er Standartmäßig "private".
Vorsicht: Ein Standartmäßig eingestellter Write-Community-String ist ein hohes Sicherheitsrisiko.
