Turnier erstellen

Mit diesem Formular, können Sie ein neues Turnier eröffnen, bzw. editieren.

Turniername

Hier können Sie den Namen des Turniers festlegen, dieser muss nicht unbedingt immer in Verbindung mit dem Spiel stehen, er kann frei erfunden werden.
Spiel
Hier müssen Sie angeben, welches Spiel in diesem Turnier gespielt wird. Es kann jedes Spiel genommen werden, bei dem mindesten ein 1on1 möglich ist.
Version des Spieles
Hier müssen Sie die Version des Spiels angeben, damit es unter den Spielern nicht zu Versiondifferenzen kommt.
Spieler pro Team
Hier legen Sie die Anzahl der Spieler fest, die das Team aufbringen muss, um am Turnier teilzunehmen.
Max. Teamanzahl
Hier können Sie die Anzahl der Teams begrenzen, die an diesem Turnier teilnehmen könnnen.
Spiel-Modus
Hier legen Sie fest, od das Turnier im Modus "Single-Elimination", einem KO Modus, gespielt wird, oder ob es in "Double-Elemination", einem Modus bei dem die "Verlierer" noch eine zweite Chance bekommen, gespielt wird.
WWCL-Modus
Hier legen Sie fest, od das Turnier ein WWCL-Turnier ist oder ein Turnier ohne WWCL-ID.
Startzeit

Hier legen Sie den ungefähren Start des Turnieres fest. Das Turnier wird dann nicht automatisch gestartet, Sie müssen es manuell eröffnen, damit sichergestellt ist, das auch alle Teams angemeldet sind, die mitspielen wollen. Dieses Turnier wird dann später in der Grafischen Übersicht sichtbar.

!!! Die Spieler können jedoch nicht an sich überschneidenden Turnieren teilnehmen. !!!
Endzeit
Hier legen Sie das ungefähre Ende des Turnieres fest. Das Turnier wird nicht automatisch abgeschlossen, Sie müssen es manuell beenden.
Dauer pro Partie
Hier legen Sie fest, wie lang eine Partie dauert, denken Sie daran, Sich mit den Partiezeiten an das ungefähre Ende das Turniers zu halten.
Altersbeschränkung
Hier können Sie bestimmen, ob das Turnier für alle Altersklassen auf der LAN offen ist, oder ob nur Spieler daran teilnehmen können, die Volljährig sind.
Bemerkungen
Hier können Sie noch ein paar Bemerkungen zu dem Turnier eintragen, wie z.B. eine Ruhepause oder ein kostenloses Mittagessen.
Regeln
Hier können Sie entweder vordefinierte Regeln auswählen, oder Sie erstellen selbst welche und tragen diese in das Formularfeld ein.
