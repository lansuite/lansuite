Messenger

If you have allready gathered experience with programms like ICQ, MSN-Messenger, or other messengers, it will surely be easy for you to understand the was lansuite-messenger works.

By the messenger it is possible to chat with other people. You allso may write messages to users currenty offline. This message will be stored in the system an be deliverd, when the user turns online again.

If you receive a message form a user not in your buddy-list, his message will be shown temorally in the group "not in list". You can add hin by clicking the + Symbol right after his username

You cann remove user from your contact list, by clicking the X an confirm the following question.

By using the button with the user-picture you can request detailed information about the user.

The message symbol in front of the username allows you to write him / her a message. If this symbol is flashing you have received a message.

The color of the username shows wether he / she is online (black), or not (red).
