Turnier Verwaltung

Mit Hilfe des folgenden Formulars können Sie ein Turnier erstellen / ändern.

Turniername

Hier geben Sie den Namen des Turniers ein.
Spiel
Hier geben Sie den Namen des Spiels ein, dass in diesem Turnier gespielt wird."
Version
Hier geben Sie die Version, in der das Spiel gespielt werden soll, ein. Dieses Feld ist optional.
Turniergruppe
Hier können Sie dieses Turnier einer Gruppe zuordnen. Ein Spieler/Team kann sich immer nur zu einem Spiel pro Gruppe anmelden.
ACHTUNG: Das nachträgliche Ändern dieses Wertes bewirkt nicht, dass bereits angemeldete Spieler von Turnieren gleicher Gruppe entfernt werden. Es verhindet nur die Teilnahme an weiteren Turnieren!
Spieler pro Team
Hier tragen Sie ein, wie viele Spieler zusammen in einem Team spielen.
ACHTUNG: Das nachträgliche Verkleinern dieses Wertes bewirkt nicht, dass bei bereits angemeldete Teams Mitglieder hinausgeworfen werden. Es verhindert nur die Aufnahme neuer Mitglieder!
Max. Teamanzahl
Hier stellen Sie ein, wie viele Teams sich maximal zu diesem Turnier anmelden dürfen.
ACHTUNG: Das nachträgliche Verkleinern dieses Wertes bewirkt nicht, dass Teams von diesem Turnier entfernt werden, falls bereits mehr Teams, als der neue Wert teilnehmen. Es verhindert nur die Aufnahme neuer Teams!
Spiel-Modus
Hier stellen Sie den Modus ein, in dem das Turnier gespielt wird. Aktuell werden 'Single Elimination' (einfaches KO-System), 'Double Elimination' und Liga unterstützt.
'Double Elimination' ist ein KO-System, bei dem Verlierer in ein sogenanntes Looser-Bracket(LB) gelangen und dort weiterspielen. Der Sieger des LB spielt gegen den Gewinner des Winner-Brackets im Finale und muss diesen dort 2x besiegen. Somit darf jeder/s der Teilnehmenden Spieler/Teams genau einmal verlieren.
Der Ligamodus ist vergleichbar mit der Modus der Fußballbundesliga.
WWCL-Spiel
Hier können Sie einstellen, ob dieses Turnier ein WWCl-Spiel ist. Die WWCL (www.wwcl.net) ist eine LanParty-Liga, zu der verschiedene LanPartys ihre Turnierergebnisse melden woraus die WWCL dann einen Gesammtsieger aller LanPartys errechnet. Der Export der Ergebnisse, zum Senden an die WWCL kann im Adminbereich herunter geladen werden und wird für jedes Turnier generiert, welches in diesem Feld hier ein WWCL-Spiel ausgewählt hat.
Die Liste der Spiele ändert sich von Zeit zu Zeit. Eine aktuelle Liste kann unter http://www.wwcl.net/download/xml_submit/gameini.xml herunter geladen werden und im Verzeichnis 'ext_inc/wwcl_games als' 'gameini.xml' abgelegt werden.
NGL-Spiel
Hier können Sie einstellen, ob dieses Turnier ein NGL-Spiel ist. Die NGL (www.ngl-europe.com) ist eine LanParty-Liga, zu der verschiedene LanPartys ihre Turnierergebnisse melden woraus die NGL dann einen Gesammtsieger aller LanPartys errechnet. Der Export der Ergebnisse, zum Senden an die NGL kann im Adminbereich herunter geladen werden und wird für jedes Turnier generiert, welches in diesem Feld hier ein NGL-Spiel ausgewählt hat.
Die Liste der Spiele ändert sich von Zeit zu Zeit. Leider beitet die NGL nicht wie die WWCL eine XML-Datei zum einfachen Import neuer Spiele an. Eine Liste mit aktuellen Spielen ist daher erst mit einem LanSuite-Update verfügbar.
Turnierbeginn
Hier können Sie den Start des Turniers angeben
Turnierende
Hier können Sie das vorraussichtliche Ende des Turnieres angeben
Dauer pro Partie
Hier stellen Sie ein, wie lange eine Partie dauert. Bei SE und DE Spielen wird hierdurch die Start- und Endzeit der jeweiligen Runde brechnet.
Altersbeschränkung
Hier können Sie für dieses Turnier eine Altersbeschränkung setzen. Ist diese gesetzt, dürfen sich nur Teilnehmer aus Sitzblöcken anmelden, welche als Ü18-Blöcke gekennzeichnet sind.
Externes Regelwerk
Hier können Sie aus einer Liste ein Regelwerk wählen. Angezeigt werden alle Dateien, die im Ordner ext_inc/tournament_rules abgelegt worden sind. Im erweiterten LanSuite-Download liegen hier standartmäßig alle WWCL- und NGL-Regeln drin.
Regeln
Hier haben Sie die Möglichkeit selbst die Regeln zu notiren, falls Sie auf keine extrene Datei zugreifen möchten.
Bemerkungen
Hier können Sie weitere Bemerkungen zu dem Turnier schreiben.
