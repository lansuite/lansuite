Moduleinstellungen

Mit diesem Formular, können Sie verschiedene Module von Lansuite aktivieren bzw. deaktivieren.

Serveradresse

Wenn der Datenbank Server auf dem gleichen Rechner, wie der Webserver läuft, geben Sie hier "localhost" an, wenn sich die Datenbank auf einem externen Rechner befindet, geben Sie hier seine Adresse an.
Danbankk
Hier tragen Sie den Namen der Datenbank ein (meist lansuite oder lansuite2).
Benutzerdaten
Hier legen Sie die Benutzerdaten für die Datenbank fest.
