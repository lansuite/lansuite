Server eintragen/editieren

Mit diesem Formular, können Sie einen Server eintragen/editieren. Die Informationen dieses Servers werden dadurch allen Benutzern zur Verfügung gestellt. Sollten sich die Serverspezifiaktionen ändern, können Sie diese dementsprechend aktualisieren.

Name

Hier können Sie den Servernamen eintragen (max. 35 Zeichen).

Typ
Hier können Sie angeben, welche Art Server Sie eintragen möchten.
IP-Adresse & Port
Hier können Sie die IP-Adresse (z.B. 10.0.100.50), die Sie am Infopoint bekommen haben eintragen.
Jenachdem, welcher Dienst auf dem Server läuft, können Sie hier den Port angeben (für gängige Dienste werden Ports vorgeschlagen), den dieser Dienst benutzt.
Betriebssytem
Hier können Sie eintragen, welches Betriebssystem auf dem Server im Einsatz ist (max. 50 Zeichen).
CPU/RAM/HDD
Hier können Sie eintragen, welche Hardware in dem Server eingebaut ist.
Passwort geschützt
Hier können Sie angeben, ob der Dienst, der auf dem Server zur Verfügung gestellt wird, mit einem Password geschützt ist oder nicht.
