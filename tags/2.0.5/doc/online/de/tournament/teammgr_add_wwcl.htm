Spieler in einem WWCL-Turnier zum Team hinzufügen

Wenn es sich bei diesem Turnier um ein WWCL Turnier handelt können nur Benutzer mitspielen, die über eine WWCL-ID verfügen.

Es gibt 2 Arten von WWCL IDs:

    Feste WWCL-IDs
    Feste IDs sind solche IDs, die einem Spieler bereits vor der LAN-Party zugewiesen wurden, d.h. wenn ein Spieler sich bereits vor der Party bei der WWCL angemeldet hat, verfügt er über eine solche ID. Wenn er diese bei Lansurfer eingetragen hat, wurde sie auch bereits automatisch importiert, und der Benutzer ist in der Liste auswählbar.
    Temporäre IDs
    Wenn der Benutzer noch nicht bei der WWCL registiert ist, hat er natürlich noch keine ID. Daher kann ihm auf der LAN-Party eine temporäre ID zugewiesen werden, die es ihm erlaubt mitzuspielen, obwohl er noch nicht bei der WWCL registriert ist. Wenn die Ergebnisse später zur WWCL exportiert werden, wird diesen Benutzern automatisch ein Konto bei der WWCL angelegt, und sie werden per eMail benachrichtigt. Hinweis: Bitte nehmen Sie zur Kentnis, dass die eMail-Adresse solcher Benutzer in diesem Falle automatisch von Lansuite an die WWCL weitergegeben wird!

Wenn einer der Spieler, die Sie hinzufügen wollen noch keine WWCL-ID hat, oder seine WWCL ID - warum auch immer - nicht importiert wurde (Sie können die WWCL-ID unter Benutzer-Details sehen) wenden Sie sich bitte an die Organisation. Diese werden Ihrem Spieler dann eine temporäre ID zuweisen. Bitte haben Sie Verständnis dafür, dass diese Zuweisung aus Sicherheits- und Verwaltungsgründen nicht automatisch erfolgen kann.
