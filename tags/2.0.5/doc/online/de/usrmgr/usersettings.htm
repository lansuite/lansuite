Pers. Einstellungen

Hier können Sie Ihre Persönlichen Einstellungen in Lansuite vornehmen.

Design
Hier können Sie zwischen verschiedenen Oberflächendesigns wählen.
Avatar

Hier legen Sie fest, welchen "Avatar" Sie benutzen wollen, dieser ist dann z.B. bei jedem Post im Forum sichtbar.

Signatur
Hier können Sie eine Signatur für Ihre Person eintragen. Diese Signatur wird dann z.B. bei jedem Post im Forum unter Ihrem Beitrag sichtbar.
